# 🎓 课程学员安装说明

## 技能名称：团队学习升级日报
**功能：** 让你的 AI Agent 每天自动分析团队状态，生成可执行的学习升级建议

---

## 第一步：下载 skill 文件

点击下方链接，下载 `.skill` 文件到你的电脑：

👉 **https://github.com/huangluckybao-lab/team-learning-upgrade-daily/raw/master/team-learning-upgrade-daily.skill**

（右键 → 另存为，保存到任意位置即可）

---

## 第二步：安装到 OpenClaw

打开你的 AI Agent 对话窗口，直接发送：

```
帮我安装这个 skill 文件：[把文件拖入对话 / 粘贴文件路径]
```

或者告诉 Agent：

```
安装 skill：/你保存的路径/team-learning-upgrade-daily.skill
```

---

## 第三步：开始使用

安装完成后，发送这句话启动：

```
帮我生成一份团队学习升级日报，
我的角色是 [老板/管理者/个人]，
可用渠道是 [微信/飞书/Discord/暂无]
```

Agent 会自动完成分析并输出报告。

---

## 常见问题

**Q：我没有团队，只有自己一个 Agent，能用吗？**  
A：可以。把"团队"理解为你自己的任务清单 + Agent 历史记录，照样能生成个人学习升级建议。

**Q：我没有绑定飞书或 Discord，怎么办？**  
A：没关系，skill 会自动检测渠道，没有外部渠道时报告会保存到本地文件，并在对话里告诉你文件位置。

**Q：用一次还是每天自动跑？**  
A：两种都行。初学者建议先手动发一次看效果，熟悉后让 Agent 帮你加一个每天定时运行的 cron 任务。
